# Seamful
> Seamful is a food delivery service app built on Rails - that allows users to order and review nearby restaurants.

</p>

## Installation

OS X & Linux:

```sh
bundle install
```

## Usage example

Seamful allows users to order and review nearby restaurants.

## Development setup

```sh
rails db:create
rails db:migrate
rails s
```
